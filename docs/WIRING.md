# AT89C51 交通灯 + 数码管 + 步进电机 + 键盘（12MHz，共阴）

电源与时钟
- VCC(5V) → MCU VCC；GND 共地
- XTAL1/XTAL2 ↔ 12MHz 晶振；两端各接 22pF 到 GND
- RST 上拉 10k 到 VCC；RST 与 GND 之间 0.1µF（上电复位）

交通灯（高电平点亮）
- P2.0 → 红灯阳极 → 串 330Ω → GND
- P2.1 → 黄灯阳极 → 串 330Ω → GND
- P2.2 → 绿灯阳极 → 串 330Ω → GND

四位共阴数码管（低电平选通）
- 段 a..g, dp（段线上各串 330Ω）：
  - P0.0→a, P0.1→b, P0.2→c, P0.3→d, P0.4→e, P0.5→f, P0.6→g, P0.7→dp
- 位选（公共阴极，低电平有效）：
  - P2.4→DIG1, P2.5→DIG2, P2.6→DIG3, P2.7→DIG4

矩阵键盘（4×4）
- 行：P3.0→R0, P3.1→R1, P3.2→R2, P3.3→R3（软件轮流拉低）
- 列：P3.4←C0, P3.5←C1, P3.6←C2, P3.7←C3（需上拉；器件通常自带）

步进电机（28BYJ‑48 + ULN2003A）
- P1.0→IN1, P1.1→IN2, P1.2→IN3, P1.3→IN4
- OUT1..OUT4 → 电机四相（按模块标注 A/B/C/D）
- COM → +5V（续流二极管公共端）
- 电机 +5V；所有 GND 共地
